An intuitive, state-of-the-art **"Culinara OS"** designed by **Ilmam I.M. (EG/2023/5646)** to revolutionize recipe management. Combining a sleek, highly responsive interface with powerful AI-driven capabilities, this project brings world cuisines to your fingertips.

Built with the distinctive **"Slate & Emerald"** design theme, featuring deep slate backgrounds with vibrant emerald and teal accents, synchronized directly with a high-fidelity Figma workspace.

---

## ✨ Hero Features

- 🤖 **AI Culinary Insights**: Integrated with **Gemini 3 Pro API** to automatically generate recipe summaries and smart meal tags.
- 📊 **Kitchen Analytics Dashboard**: Real-time visual insights using **Chart.js** displaying cuisine distribution, difficulty levels, and prep time analysis.
- ⚡ **Optimized UX & Animations**: 60fps-capable micro-interactions, smooth slide-over panels for recipe details (ingredients & instructions), and fluid skeleton loaders.
- 💾 **Chef Profile Management**: Full professional settings suite with Base64 profile picture persistence and real-time global state hydration via Pinia.
- 📡 **Live Data Engine**: Consumes the public **DummyJSON Recipes API**, mapping world-class culinary data into a premium user experience.

---

## 🛠 Tech Stack

- **Framework**: Vue 3 (Composition API)
- **Styling**: Tailwind CSS (Custom Design System Tokens)
- **State Management**: Pinia
- **Data Source**: DummyJSON Recipes API
- **AI Integration**: Google Gemini 3 Pro API
- **Data Visualization**: Chart.js / vue-chartjs
- **Build Tool**: Vite

Culinara OS represents a peak synthesis of modern frontend engineering and intuitive culinary experience design. # 4. Launch Development
npm run dev
